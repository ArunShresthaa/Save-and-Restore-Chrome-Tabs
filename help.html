Tab Exporter Help

Welcome to Tab Exporter! This extension helps you manage your Chrome tabs more efficiently.

Features:

  1. Save: Saves tabs in the internal memory of Chrome, no file is downloaded.
  2. Restore: Restores recent tabs from the internal memory of Chrome.
  3. Download: Downloads tabs in a text file with the given file name.

How to Use:

  1. Click the Save Tabs button to save your current tabs in Chrome's internal memory.
  2. Click the Restore Tabs button to restore your recently saved tabs.
  3. Click the Download Tabs button to download a text file with the URLs of your open tabs.
  4. Use the "?" button to access this help page anytime.
